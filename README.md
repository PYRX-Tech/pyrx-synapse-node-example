# Synapse Node.js Example

Example project showing how to use [@pyrx/synapse](https://www.npmjs.com/package/@pyrx/synapse) with Node.js.

## Setup

1. `npm install`
2. Copy `.env.example` to `.env` and fill in your API key
3. Run an example:

```bash
npm run track      # Track an event
npm run identify   # Identify a contact
npm run send       # Send a transactional email
```
